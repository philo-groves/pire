# Parity Weave Live Lab

Small local target for live PiRE improvement sessions. This runnable lab exercises:
- larger static analysis across three bundle files with exact-cardinality requirements
- decoy local optima where a compact synthetic bundle can satisfy part of the state but not the full proof gate
- non-obvious proof flow spread across parsing, weave reduction, and global consistency checks
- disclosure-to-proof carryover with target-created proof
